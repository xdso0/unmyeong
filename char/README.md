# ✅ 앱이 실제 쓰는 36명 (이것부터!)
헤드라인의 **"운명 캐릭터 팀"** = 사주 × 자미두수 조합으로 한 사람당 2~3명을 보여줘:
- **타고난 나(일간 = 천간 10)** — 항상
- **관계의 뿌리(일지 = 지지 12)** — 항상
- **인생 무대(명궁 = 자미 주성 14)** — 시(時) 있을 때

→ **명궁14 + 천간10 + 지지12 = 36장.** 없는 칸은 색 배지(글자)로 폴백되니 하나씩 채워도 OK.
