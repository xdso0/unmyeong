# 사주 × 자미두수 운명분석

동양의 **사주(오행·십성)** + **자미두수(명궁·12궁·14주성)** + 서양 **별자리**를 한 번에 계산해
성격·운명의 상대·피해야 할 상대·궁합을 풀어 주는 **단일 HTML 웹앱**.

- 🌐 **라이브:** https://xdso0.github.io/unmyeong/
- 📱 모바일 우선 · 글래스(초록·파랑) 디자인 · 동네 언니 반말 톤
- 🔒 **개인정보는 전부 브라우저(localStorage)에만 저장** — 외부 전송 없음

---

## 📂 구조 (수정하는 법)

이 앱은 **`index.html` 파일 하나**가 전부예요. (CSS·JS·데이터 모두 안에 들어 있음)

> ⚠️ **이 저장소의 `index.html`은 빌드 산출물입니다 — 직접 수정 금지.**
> 정본은 상위 폴더의 `../사주별자리_운명분석.html` 하나뿐이에요.

수정하려면 (상위 폴더에서):
1. `사주별자리_운명분석.html`(정본)을 편집기로 열어서 고친다
2. `python build.py --bump` 실행 → BUILD 태그 +1, 압축된 `site/index.html` 자동 생성
3. `cd site && git add -A && git commit -m "수정 내용 BUILD <태그>" && git push`

자세한 규칙은 `../README.md` 참고.

> 로컬 미리보기: 상위 폴더에서 `python -m http.server 8744` 실행 후
> `http://localhost:8744/site/index.html` 열기.

### index.html 안의 주요 블록 (Ctrl+F로 찾기)

**계산 엔진**
- `pillarsFromDate()` — 사주 4기둥(년·월·일·시) 계산
- `sunApparentLong()` / `ipchunJDE()` — **절기 천문 정밀 계산**(태양 황경, 분 단위 절입)
- `solarToLunar()` — 음력 변환(KASI 호환 데이터)
- `ziwei()` / `renderZiwei()` — **자미두수** 명궁·오행국·자미성·14주성·사화·대운
- `sipsungCounts()` / `strengthInfo()` / `yongsin()` — 십성·신강신약·용신
- `compatibility()` / `ziweiCompat()` — 궁합(사주+별자리+자미 교차)

**화면 렌더**
- `describePerson()` — 종합 풀이
- `renderMyDetail()` — '나의 분석'(목차 아코디언 + 융합 박스)
- `renderFate()` / `renderAvoid()` — 운명의 상대 / 피해야 할 상대
- `renderCompatList()` — 궁합 8장 + 자미 관계운
- `saveResultImage()` — 인스타 공유 카드(1080×1920)

**데이터 테이블** (오행별 문구 — 여기 고치면 내용이 바뀜)
- `EL`, `SIPSUNG`, `EL_PERSONA`, `AVOID_DETAIL`, `PARTNER_EXTRA`, `ZW_STAR`, `ZODIAC` 등

## 🧮 계산 방식 & 근거 (신뢰 출처)

명식(命式) 계산은 천문·표준 알고리즘 기반이라 **검증 가능**합니다.
(풀이/해석은 전통 명리·점성술 기반의 **오락·자기성찰용** 콘텐츠로, 과학적 검증 대상이 아닙니다.)

| 계산 | 방식 | 근거 |
|---|---|---|
| 일주(60갑자) | 율리우스적일(JDN) 연속 계산 | 위키 *Sexagenary cycle* (1949-10-01=甲子 검증) |
| 월주 | **12절기**(태양 황경 15°) 기준, 분 단위 절입 | Jean Meeus *Astronomical Algorithms* / VSOP87 |
| 연주 | 입춘 절입 기준 | 표준 명리 역법 |
| 시주 | 시지 + 오서둔(五鼠遁) | 고전 |
| 음력 | 1900–2100 역법 데이터 | 한국천문연구원(KASI) 호환 |
| 자미두수 | 명궁·오행국(납음)·자미 안성결·14주성·사화·대운 | 자미두수 고전 안성결 |

> 한계: 신강신약 판정·용신·묘왕평함·일부 사화는 명리/자미 **유파마다 다름**(앱에 "통용 기준" 표기). 절기는 천문 계산으로 분 단위까지 정밀화했으나 KASI 공식 절입시각과 수 분 차이가 날 수 있음.

## 🔒 개인정보 · 보안

- 이름·생년월일·정체성 등 **입력값은 localStorage(이 브라우저)에만** 저장, 외부 전송 0.
- 저장소(GitHub)에는 **앱 코드(index.html)만** 있고 개인정보 없음.
- 백업은 앱 내 JSON 내보내기로. 선택적 PIN 잠금 있음.
- 실사 사진 모드(선택): 추정 인물 이미지만 외부 생성 서버에 **오행 키워드+seed**만 전송(개인정보 미전송).
- 커밋 작성자 이메일은 noreply로 설정됨.
